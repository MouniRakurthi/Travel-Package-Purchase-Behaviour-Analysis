# ✈️ Travel Package Purchase Behaviour Analysis

## 📌 Overview

This project focuses on analyzing customer purchase behaviour for travel packages offered by a travel company. The objective is to identify the factors influencing travel package purchases and understand customer segments with higher conversion potential.

The project follows an end-to-end data analysis workflow, including data cleaning, exploratory data analysis (EDA), hypothesis testing, dashboard development, and business reporting to support data-driven decision-making.

---

## 📁 Dataset

The dataset contains customer information, travel preferences, sales interaction details, and package purchase status.

### Key Features

* Customer Demographics (Age, Gender, Marital Status, Occupation, City Tier)
* Financial Information (Monthly Income)
* Sales Interaction Details (Type of Contact, Duration of Pitch, Number of Follow-Ups)
* Travel Behaviour (Number of Trips, Passport, Preferred Property Rating)
* Product Information (Travel Package Offered)
* Purchase Status (ProdTaken)

The dataset was cleaned and transformed before analysis to ensure consistency and accuracy.

## 📈 Results & Insights

* Overall package conversion rate is 18.8%.
* Product package type significantly influences customer conversion.
* Customer contact method impacts purchase behaviour.
* Age and income levels are associated with package purchases.
* Marital status and occupation influence conversion patterns.
* Longer sales pitches are associated with higher conversion rates.
* Statistical testing validated key business findings.
